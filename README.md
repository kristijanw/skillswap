## About

**SkillSwap** is a mobile app that connects people who want to exchange knowledge and skills — no money, no middlemen. If you play guitar and want to learn programming, SkillSwap will match you with someone who does exactly that and wants to learn music in return.

The platform is built around a local learning community where every user can be both a teacher and a student at the same time.

## How it works

1. **Create a profile** — add the skills you can teach and the ones you want to learn
2. **Discover people** — browse profiles and like the ones that interest you
3. **Match** — when you both like each other, a chat opens up
4. **Make a deal** — arrange your skill exchange however works best for you
5. **Work portfolio** — showcase what you know through photos and short videos

## Features

- Sign in with email or Google account
- Swipe interface for discovering users
- Real-time chat between matches
- Work portfolio with up to 5 images and a video up to 1 minute
- Trust levels for verified users
- Report system for inappropriate content
